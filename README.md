# String Similarity Engine

A comprehensive string similarity comparison tool with multiple metrics and an interactive Streamlit interface.

---

##  Features

-  **Levenshtein Distance** — character-level edit similarity  
-  **Jaccard Similarity** — word overlap comparison  
-  **Cosine Similarity** — vector-based similarity  
-  **Semantic Similarity** — transformer-based embeddings  
-  **Interactive Visualizations** — radar charts, breakdowns, and insights  
-  **Real-time UI** powered by Streamlit  

---

##  Tech Stack

### Core Technologies

| Category | Technology | Purpose |
|--------|-----------|--------|
| Language | Python | Core programming language |
| Web Framework | Streamlit | Interactive UI and real-time visualization |
| Deep Learning | PyTorch | Backend for transformer models |
| Transformer Models | Sentence-Transformers | Semantic embedding generation |
| NLP Library | Hugging Face Transformers | Access to pre-trained models |
| Scientific Computing | NumPy | Numerical operations |
| Data Handling | Pandas | Data manipulation |
| Machine Learning | Scikit-learn | TF-IDF & cosine similarity |
| Visualization | Plotly | Charts and visual analytics |
| String Matching | Python-Levenshtein | Fast edit distance |
| Text Processing | NLTK | NLP utilities |
| Scientific Tools | SciPy | Advanced computations |
